前置詞（小テスト）

下の空欄に適した語（前置詞など）を入れてください。
  1. She is tall ( ) Japanese.
    ※「日本人にしては背が高い」という意味
  2. Make sure to submit this form ( ) two weeks.
  3. Make sure to submit this form ( ) tomorrow.
  4. He can fix anything ( ).
    ※「彼は何でも簡単に直せる」という意味
  5. I can eat anything ( ) pineapples.
  6. ( ) the summer vacation, all the books in BookOFF were on sale.
  7. Bill Gates told us a key ( ) making a fortune.
  8. The price of Tesla's shares dropped ( ) four thousand dollars.
  9. I can't stop dancing ( ) the music.

模範解答

  1. She is tall ( for a ) Japanese.
  2. Make sure to submit this form ( within ) two weeks.
  3. Make sure to submit this form ( by ) tomorrow.
  4. He can fix anything ( with ease ).
  5. I can eat anything ( but ) pineapples.
  6. ( During ) the summer vacation, all the books in BookOFF were on sale.
  7. Bill Gates told us a key ( to ) making a fortune.
  8. The price of Tesla's shares dropped ( by ) four thousand dollars.
  9. I can't stop dancing ( to ) the music.
